# 🏭 台灣工廠化學品管理系統

> 整合台灣職安法規的開源化學品管理工具，適用於各類製造業工廠的職業安全衛生人員使用。

[![License: MIT](https://img.shields.io/badge/License-MIT-green.svg)](LICENSE)
[![GitHub Stars](https://img.shields.io/github/stars/dogmingeatmygg/chemical?style=social)](https://github.com/dogmingeatmygg/chemical)

---

## ✨ 功能特色

- 🔍 **輸入 CAS NO 自動判讀**法規類別，不需手動查表
- 📋 **整合 8 大法規資料庫**，共 2,040 筆資料：
  - 優先管理化學品（第1/2/3款，1,063筆）
  - 消防危險物品管制量（104筆）
  - 有機溶劑分類（79筆）
  - 特定化學物質（79筆）
  - 毒化物（366筆）
  - 管制性化學品（21筆）
  - 空氣有害物容許濃度（285筆）
  - 特殊健檢附表十（43筆）
- 🏢 **多部門管理**：一個化學品可對應多個使用部門
- 🧪 **混合物支援**：一個產品可輸入多個 CAS 成分
- 📊 **一頁式卡片**：不需左右滾動，所有資訊一目瞭然
- 📥 **匯出 Excel**：可匯出產品清單、成分明細、法規 DB
- 🔒 **管理員/唯讀 雙模式**：同事只能查閱，不能修改
- 💾 **資料存在自己的 GitHub**：完全掌控，別人看不到

## 🚀 快速開始（約 20 分鐘）

### 第一步：Fork 此專案

點右上角 **Fork** 按鈕，複製一份到您自己的 GitHub 帳號。

### 第二步：修改設定

Fork 後，在您自己的儲存庫中點開 `index.html`，找到第 8 行：

```javascript
const GITHUB_USER = 'YOUR_USERNAME';   // ← 改成您自己的 GitHub 帳號
const GITHUB_REPO = 'chemical';        // ← 通常不需要改
```

將 `YOUR_USERNAME` 換成您的 GitHub 帳號名稱後儲存。

### 第三步：開啟 GitHub Pages

1. 進入您的儲存庫 → **Settings** → **Pages**
2. Source 選 **Deploy from a branch**
3. Branch 選 **main** → **Save**
4. 等約 1 分鐘，網址即生效：`https://您的帳號.github.io/chemical/`

### 第四步：建立 Personal Access Token（管理員功能用）

1. GitHub 右上角頭像 → **Settings**
2. 左側最底部 → **Developer settings**
3. **Personal access tokens** → **Tokens (classic)**
4. **Generate new token (classic)**
5. 勾選 **repo** 權限，建立後**立刻複製儲存**（只顯示一次）

### 第五步：開始使用

開啟您的網址，右上角點「**管理員登入**」，輸入 Token 即可新增、編輯化學品資料。

## 📖 使用說明

### 新增化學品

1. 管理員模式下點右上角「＋ 新增化學品」
2. 填入產品編號（可自訂格式，如 `A-001`）、名稱、閃火點、水溶性
3. 填入各成分的 **CAS NO**，系統自動帶出法規判讀結果
4. 儲存後自動同步到 GitHub，所有人立即看到最新資料

### 查閱資料（同事）

直接開啟網址即可，不需要帳號，唯讀模式，無法修改任何資料。

### 匯出 Excel

管理員模式下點「📊 匯出 Excel」，可選擇：
- 全部產品清單
- 目前選中的產品（含成分和部門）
- 法規 DB 任一子表

### 更新法規 DB

當本專案發布新版本（Release），下載新的 `index.html` 上傳覆蓋即可獲得最新法規資料。
